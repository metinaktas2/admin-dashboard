# 📊 Admin Dashboard App

A fully responsive admin dashboard application designed to manage products, users, and orders with interactive data visualization.

This project provides a clean and intuitive interface for monitoring business data and managing e-commerce operations efficiently.

---

## 🚀 Features

* 📦 Product management (list, edit, add)
* 🧾 Order tracking with delivery status
* 👥 User activity and order insights
* 📊 Data visualization with charts
* 📱 Fully responsive dashboard layout
* 🧭 Sidebar navigation for seamless page transitions

---

## 🛠️ Tech Stack

* **JavaScript (React)**
* **json-server** (mock backend with REST API)
* **Chart.js**
* **react-chartjs-2**
* **react-icons**
* **react-toastify**

---

## 📊 Dashboard & Analytics

* Products are displayed in a **card-based layout**
* Category-based analytics are visualized using:

  * 📈 Line charts
  * 🥧 Pie charts

These charts provide quick insights into product distribution and trends.
